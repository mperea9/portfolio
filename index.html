Student Project: Pathfinding Visualizer

University of Colorado - Boulder

This is a path finding visualizer tool that helps someone visualize how the A* search algorithm works in real time as it runs, implemented using Python. The A* Search Algorithm is an extension of Dijkstra's algorithm with Breadth-First characteristics. The A* search also introduces a heuristic into a regular graph-searching algorithm, essentially planning ahead at each step so a more optimal decision is made, making it one of the more efficient methods in pathfinding and graph traversal. There are two heuristics of this algorithm, the Manhattan Distance and Euclidean Distance. This implementation uses the Manhattan Distance heuristic.
